
# ENGLISH FLASHCARD WEBSITE

Website learning English via falshcard

## Guideline
Follow below instruction to setup project at local
### 1. Clone repo
``` bash
git clone https://github.com/lamnguyentuan/PKL_English.git
cd PKL_English
```
#### 2. Create virtual enviroment

```bash
python -m venv venv
venv\Scripts\Activate
```
### 3. Install dependencies
```bash
pip install -r requirements.txt
```
